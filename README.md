# 🏥 MediGuide AI

MediGuide AI is a full-stack AI-powered healthcare assistant designed to help users understand symptoms, explore trusted medical information, analyze medical reports, and discover nearby healthcare facilities using real-world data.

## ✨ Vision

Build a one-stop AI healthcare companion that combines:
- AI-powered health conversations.
- Symptom guidance and education.
- Home remedies and wellness suggestions for mild conditions.
- Medical report summarization.
- Nearby hospital and doctor discovery.
- Personal health history management.

## 🛠️ Planned Technology Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- FastAPI
- Python
- LangGraph
- Ollama

### AI & Data
- Local Open Source LLMs
- ChromaDB
- PostgreSQL
- Retrieval-Augmented Generation (RAG)

### Maps & Healthcare Data
- OpenStreetMap
- Overpass API
- Leaflet

## 🎯 Project Goals

- 100% free-first development.
- No mandatory AI API subscriptions.
- Production-quality architecture.
- Resume and placement ready.
- Modern, scalable, and maintainable codebase.

## 📁 Project Structure

- `frontend/` — Next.js frontend.
- `backend/` — FastAPI backend and AI agent.
- `docs/` — Project documentation.
- `assets/` — Images and static assets.
